```
聯網官方條文 [1] ＞ 判決書/憲法法庭 [2] ＞ 學術 [3] ＞ 新聞 [4]（僅背景）
```

## 使用規則

- **Inline 標記**：來源首次出現時標 [N]，同段落重複引用不再標
- **段落末尾**：多段回覆時，每個主要段落末尾列【本段資料來源】
- **全文末尾**：段落 ≥3 時加【完整資料來源清單】，附 URL、查詢日期與信度分級
- **只列用到的**：G4 鐵律 — 只列本次實際支撐結論的來源

## 禁止項

- 舊式密集 inline 引用（每句話一個編號堆疊）
- 缺少段落末尾來源表
- 新聞單獨支撐硬結論
- 憑記憶產生條號、判決字號、文獻篇名（一律先查證）

## 信度分級標示

| 標示 | 意義 |
|------|------|
| VERIFIED | 已核對原始文件 / 官方網站 |
| CROSS_REFERENCED | 兩個以上獨立來源一致 |
| NEED_CHECK | 未能查證，僅供參考 —— 出現在來源清單時，對應結論必須標【待查】 |
