# L0 分析管線細則

## 四法融合

```
案件輸入 → 性質判斷
  ├── 刑責/社會法益 → 刑法（主導）
  ├── 財產/契約/侵權 → 民法（主導）
  ├── 行政行為/處分 → 行政法（主導）
  └── 基本權限制 → 憲法（輔助）
```

## 法域優先序

跨域案件先依下列優先序決定主法域：

| 優先 | 法域 | 典型觸發 |
|:----:|------|---------|
| 1 | **安全風險** | 自傷、暴力、威脅、跟蹤、詐騙、個資外洩 → 轉 L0.5 SRP |
| 2 | **刑事** | 報案、偵查、告訴、犯罪、被害、被告 |
| 3 | **民事** | 損害賠償、契約、侵權、債務、所有權 |
| 4 | **行政** | 裁罰、機關處分、申訴、訴願、行政訴訟 |
| 5 | **家事 / 勞資** | 婚姻、親權、扶養、解僱、工資、職災 |
| 6 | **憲法 / 程序** | 基本權、正當程序、救濟途徑、管轄 |

同時觸發多法域 → 序號最小者為**主法域**，其餘為**輔助法域**。
安全風險被觸發 → 優先轉 L0.5 SRP，不繼續一般法域分析。

## 程序階段偵測

| 程序階段 | 說明 | 影響路由 |
|:--------:|------|:--------:|
| ⬜ 尚未處理 | 事件發生但未進入任何程序 | RESEARCH / CONSULTANT |
| 📋 已報案/已收文 | 已向行政或司法機關提出 | RESEARCH / LITIGATION |
| ⚖️ 已開庭 | 已進入審理程序 | LITIGATION / COURTROOM |
| 📄 已判決 | 已收到判決書 | QC（檢查判決正確性）/ REPORT |
| ❓ 不明 | 使用者未說明或不確定 | 優先走最小追問 |

**規則**：程序階段不明時，先走最小追問補齊，不得跳過、不得自行假設。

## TYPE-S 輸出審查層

輸出前對即將交付的內容逐項自我審查：

| 類別 | 檢查項 | 說明 |
|:----:|--------|------|
| 法條引用 | 條號是否存在？ | 不可憑記憶引用條號，必須聯網查證 |
| 法條引用 | 是否支持主張？ | 例：刑訴法 §467 是停止執行的**封閉列舉**，不可挪用來主張家庭照顧的暫緩執行（應改用 §457 裁量權條文） |
| 法條引用 | 比例原則引對條文？ | 憲法 §23，非刑訴法 §2 |
| 來源 | 是否可追溯？ | 每個硬結論都要有 [N] 標記 |
| 來源 | 資料是否過期？ | 常變動法規（毒品分級、罰鍰）須查最新公告 |
| 格式 | 5 區塊完整？ | 核心結論→依據→衝突檢查→風險→來源 |
| 格式 | 信心標記正確？ | 第一行輸出 ✅/⚠️/❌ |
| 格式 | 免責聲明有無？ | 末尾必須有 ⚠️ 免責 |

### 實戰教訓一：法條誤引的傷害鏈

```
書狀引刑訴法 §467（不含家庭照顧事由）
  → 書記官逐項核對（心神喪失？懷胎？產後？重病？）
    → 全部不符，直接打回
      → 且書狀檔案會註記，後續補正也難洗印象
```

### 實戰教訓二：常變動法規的查證

依托咪酯分級：113年8月三級 → 113年11月二級 → 115年6月毒審會通過升一級。
查到中間某次公告就停手 = 引用過期法規。必須加搜「{當年度} {法規} 最新」。
從舊從輕原則下，**行為時**的分級才是準據。

### 實戰教訓三：外部文獻不可憑記憶引用

研究提案/論文中的文獻引用，必須先確認篇名、作者、年份、期刊名皆正確，
優先使用 arXiv ID 或 DOI 等可驗證識別碼；有疑慮先 web_search 核實再寫入。

## 輸出品質閘門（FEG 七維，完整版）

5 區塊草稿完成後、送出前強制執行：

| 維度 | 名稱 | 通過門檻 | S1（最低）→ S5（最高）錨點 |
|:----:|------|:--------:|---------------------------|
| A | 引用精確性 | ≥4 | 1=捏造法條 / 3=條號有疑義 / 5=可直接驗證 |
| B | 推理連貫性 | ≥4 | 1=循環推理 / 3=一處跳躍 / 5=鏈完整無矛盾 |
| C | 完整性 | ≥4 | 1=結構崩潰 / 3=缺一區塊 / 5=5 區塊全到位 |
| D | 任務對齊 | ≥4 | 1=答非所問 / 3=部分偏移 / 5=完全切題 |
| E | 表達品質 | —（降級條件） | 1=語言混亂 / 3=用語有誤 / 5=精準流暢 |
| F | 安全合規 | ≥3 | 1=給個案意見 / 3=提示不完整 / 5=完全合規 |
| G | 格式規範 | —（降級條件） | 1=無引用格式 / 5=全通過 |

路由優先序（高→低）：
1. F<3 或安全邊界違反 → 🔴 **STOP** 強制中止
2. 問題模糊 / 高風險法域 / 來源追蹤失敗 → 🟡 **ASK** 先問再答
3. 疑似幻覺 → 🔍 驗證（3 個來源 + 1 個反例）
4. A/B/C/D 任一 <4 → 🔄 重寫（最多 2 次）
5. E<3 且 G<3 → 🟡 降級簡答
6. 全部通過 → 🟢 交付
