# ⚖️ 引用政策

## 核心原則

> **每條法條、每則判決、每段推論，都必須有可驗證的來源。**

## 引用格式

```
[T]台灣高等法院台中分院112年度上易字第123號民事判決
[1]民法第184條第1項前段 — 侵權行為之一般成立要件
[2]最高法院105年度台上字第89號民事判決 — 故意過失之認定
[3]王澤鑑，《侵權行為法》，2020年，第89-92頁
```

### 代碼說明

| 代碼 | 含義 |
|:-----|:------|
| `[T]` | 判決字號（直接引用） |
| `[1]` | 法條條文（附條文內容） |
| `[2]` | 判決先例（支持性引用） |
| `[3]` | 學說見解（輔助性引用） |

## 禁止事項

| ❌ 禁止 | ✅ 替代 |
|:--------|:--------|
| 虛構法條條號 | 查詢後如無法驗證，標記 `[待查]` |
| 憑印象引用判決字號 | 改引可驗證的判決，或標註 `[推論]` |
| 盲信 LLM 生成的引用 | 所有引用必須經 RAG 或網路查證 |
| 使用過期的法條版本 | 確認最新修法日期後引用 |

## 驗證流程

```
引用 → RAG 查詢 → 法條比對 → 判決查核 [judicial.gov.tw] → 學說確認 → 輸出
  │         │           │            │                     │
  原始      SQLite     原文比對    司法院裁判書查詢       律師文章/教科書
```
