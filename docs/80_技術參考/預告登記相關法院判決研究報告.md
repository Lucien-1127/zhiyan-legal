## 一、預告登記塗銷請求（民法§767）

### 判決：最高法院95年度台上字第2323號民事判決

| 項目 | 內容 |
|:-----|:------|
| **裁判字號** | **最高法院95年度台上字第2323號** |
| **案由** | 塗銷預告登記事件 |
| **裁判日期** | 2006年10月26日 |
| **關鍵爭點** | 預告登記同意書得否撤銷？所有權人依民法§767請求塗銷之要件 |
| **實務見解** | 若預告登記所保全之請求權已不存在（如買賣契約解除），登記名義人得依民法§767請求塗銷；但若基於有效同意書辦理，不能逕以同意書得撤銷而主張塗銷 |

## 二、預告登記與請求權時效（民法§125）

### 判決：最高法院103年度台上字第1038號民事判決

| 項目 | 內容 |
|:-----|:------|
| **裁判字號** | **最高法院103年度台上字第1038號** |
| **案由** | 塗銷預告登記等 |
| **裁判日期** | 2014年6月5日 |
| **關鍵爭點** | 預告登記所保全請求權之消滅時效；時效完成後預告登記之效力 |
| **實務見解** | ①預告登記所保全請求權之時效依原請求權性質定之（一般為§125之15年）；②預告登記本身不延長時效；③時效完成後非經塗銷程序預告登記仍存在；④時效抗辯成立時登記名義人得請求塗銷 |

## 三、預告登記被詐欺撤銷（民法§92/§93）

### 判決：最高法院92年度台上字第1309號民事判決

| 項目 | 內容 |
|:-----|:------|
| **裁判字號** | **最高法院92年度台上字第1309號** |
| **案由** | 確認本票債權不存在等 |
| **裁判日期** | 2003年6月19日 |
| **關鍵爭點** | 因受詐欺而簽署預告登記同意書，得否依民法§92撤銷？ |
| **實務見解** | ①預告登記同意書之簽署為法律行為；②若因被詐欺或脅迫而為同意，得依民法§92於發現詐欺後一年內（§93）撤銷；③撤銷後同意書自始無效，預告登記失所附麗；④請求權人如有詐欺行為，可能另涉刑法§339詐欺罪。實務上常見於民間二胎借款時，金主/代書利用借款人急迫輕率，使其以預告登記方式擔保非真實之債權 |

## 四、預告登記與借名登記糾紛

### 判決：最高法院108年度台上字第2162號民事判決

| 項目 | 內容 |
|:-----|:------|
| **裁判字號** | **最高法院108年度台上字第2162號** |
| **案由** | 請求移轉土地所有權等 |
| **裁判日期** | 2019年9月19日 |
| **關鍵爭點** | 借名登記關係消滅後，借名人得否請求塗銷出名人擅自設定之預告登記？ |
| **實務見解** | ①借名登記中出名人僅為名義登記人；②若出名人未經借名人同意擅自同意第三人辦理預告登記，而第三人明知或可得而知借名登記關係，該預告登記可能因違反誠信原則或構成權利濫用被認定無效；③借名人得終止借名登記關係後請求返還土地並塗銷預告登記 |

## 五、代書／地政士違法辦理預告登記之刑事責任（刑法§214）

### 判決：臺灣高等法院110年度上易字第1167號刑事判決

| 項目 | 內容 |
|:-----|:------|
| **裁判字號** | **臺灣高等法院110年度上易字第1167號** |
| **案由** | 偽造文書 |
| **裁判日期** | 2021年8月17日 |
| **關鍵爭點** | 地政士明知預告登記同意書非本人親簽，仍持向地政機關申辦，是否構成刑法§214使公務員登載不實罪？ |
| **實務見解** | ①地政士辦理土地登記時對申請文件真正與否負有實質審查義務（地政士法§18、§22）；②若明知登記原因文件不實仍持向地政機關申辦，構成刑法§214（三年以下有期徒刑）；③可能另涉刑法§215業務上文書登載不實罪、§216行使偽造文書罪、§339詐欺罪 |
