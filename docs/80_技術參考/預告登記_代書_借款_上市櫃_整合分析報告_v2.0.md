## 壹、土地預告登記制度

### 一、法律框架

土地法第79條之1第1項規定三類可保全之請求權，須由請求權人檢附所有權人同意書辦理^[全國法規資料庫]。第2項明定：預告登記未塗銷前，有妨礙所登記請求權之處分無效。第3項例外：徵收、法院判決、強制執行不受限制^[全國法規資料庫]。

**內政部函釋補充**：內政部108年7月11日台內地字第1080263887號函簡化預告登記申請程序，得免附登記清冊^[內政部108年函釋]。另內政部營建署110年8月25日內授營土字第1100813960號函就合宜住宅預告登記限制期間屆至塗銷後得否加入社會住宅包租代管計畫為出租住宅為解釋^[內政部110年函釋]。

### 二、效力定性 — 債權物權化

學說上將預告登記定性為「債權物權化」之典型——本質為債權之請求權，經登記後取得一定對世效力。此與民法第425條買賣不破租賃、第826條之1分管契約屬同一類型討論^[民法§425][民法§826-1]。

**學者見解**：

王澤鑑教授於《民法物權》及《民法學說與判例研究》中指出，預告登記係德國Vormerkung理論之繼受，為債權物權化三種類型之一（另二者為買賣不破租賃與分管契約），效力上採**處分相對無效說**^[王澤鑑，《民法物權》；《民法學說與判例研究》]。

謝在全教授於《民法物權論》（上中下）中詳述物權法定主義與預告登記之交錯，採**實務導向分析**，其為民法物權編修正之主要領導人^[謝在全，《民法物權論》]。

黃茂榮教授於《債法總論》及《法學方法與現代民法》中強調：債權物權化非真正物權化，側重債權確保功能與稅法交錯，方法論一貫^[黃茂榮，《法學方法與現代民法》]。

温豐文教授於《土地法》（2007年，頁209）自土地登記制度定位，分析預告登記之防患未然功能^[温豐文，《土地法》]。

陳立夫教授於《土地法研究》及相關期刊論文中，就地政專業視角、實務運作爭議、立法沿革考證有深入論述^[陳立夫，《土地法研究》]。

**共通見解**：預告登記為債權物權化典型類型，效力上採**處分相對無效說**。

### 三、時效議題

預告登記本身是否因時間經過而消滅，實務與學說有不同見解：

**最高法院103年度台上字第1038號民事判決**（2014.06.05）為此問題之指標裁判：預告登記所保全之請求權，其消滅時效仍依原請求權之性質定之（一般為民法第125條之15年）；預告登記本身不延長時效期間。但預告登記未塗銷前，登記名義人對外處分土地之行為對請求權人有妨礙者無效（土地法第79條之1第2項）。因此，縱使請求權時效已完成，非經塗銷程序，預告登記仍繼續存在；然若時效抗辯成立，登記名義人得據此請求塗銷預告登記^[最高法院103年度台上字第1038號判決]。

### 四、塗銷路徑

實務上主要有四種方式：①權利人出具同意書（最簡便）；②所有權人提起確認請求權不存在之訴或依民法§767請求除去^[民法§767]；③請求權因清償等事由消滅後請求塗銷；④徵收、判決、強制執行可穿透^[土地法§79-1③]。

**最高法院95年度台上字第2323號民事判決**（2006.10.26）確立：若預告登記所保全之請求權已不存在（如買賣契約解除），登記名義人得依民法§767請求塗銷；但若預告登記係基於有效之同意書辦理，則不能逕以同意書係得撤銷而主張塗銷^[最高法院95年度台上字第2323號判決]。

**內政部107年9月11日台內地字第1070064039號函**補充：已辦竣預告登記之土地如欲申辦信託登記，仍須先塗銷預告登記後始得辦理信託登記，不得持憑預告登記請求權人同意書逕行辦理信託登記，確立預告登記與信託登記之先後順序^[內政部107年函釋]。

**法務部110年8月24日法律字第11003508250號函**就監護人代理塗銷預告登記是否須經法院許可為詳細分析：塗銷預告登記常併同不動產權利移轉登記，是否須經法院許可宜綜合判斷^[法務部110年函釋]。

## 貳、代書（地政士）的法律框架

### 一、法定義務框架

地政士法第18條：應查明委託人確為登記標的物之權利人或權利關係人，並核對其身分^[地政士法§18]。第22條：簽證時應查明簽訂人身分為真正，簽證不實或錯誤致生損害者負賠償責任^[地政士法§22]。第27條第6款：禁止明知為不實之權利書狀、印鑑證明或其他證明文件而向登記機關申辦登記^[地政士法§27-VI]。

### 二、刑事責任框架

刑法第214條（使公務員登載不實罪）法定刑為3年以下有期徒刑^[刑法§214]。第215條（業務登載不實）、第339條（詐欺，5年以下）、第342條（背信，5年以下）^[刑法§339][刑法§342]。各條之成立以「故意」為要件，過失不罰。

**臺灣高等法院110年度上易字第1167號刑事判決**（2021.08.17）為地政士刑事責任之重要參考：地政士（代書）辦理土地登記業務時，對於申請登記之文件是否真正、有無經所有權人同意，負有實質審查義務（地政士法§18、§22參照）。若明知登記原因文件不實（如預告登記同意書係偽造或未經本人同意），仍持向地政機關申辦登記，使地政人員將不實事項登載於土地登記簿，構成刑法§214使公務員登載不實罪，可處三年以下有期徒刑。此外，尚可能涉及刑法§215業務上文書登載不實罪、§216行使偽造文書罪及§339詐欺罪^[臺灣高等法院110年度上易字第1167號刑事判決]。

### 三、民事責任框架

地政士法第26條：不正當行為或違反業務義務致生損害負賠償責任^[地政士法§26]。第22條：簽證不實由簽證基金代為支付，額度依管理辦法規定^[地政士簽證責任及簽證基金管理辦法]。民法第184條侵權行為、第185條共同侵權行為連帶責任，以故意或過失為要件^[民法§184][民法§185]。

### 四、行政責任框架

地政士法第43條：懲戒處分種類——警告、申誡、停止執行業務（2月~2年）、除名^[地政士法§43]。第44條列舉構成懲戒之情事，由地政士懲戒委員會審議^[地政士法§44]。

## 參、代書貸款情境下的三性觀察

以下觀察以「債務人無實際借款」為前提，整理法律框架下可能涉及的討論面向。

### 一、制度目的面向（合理性）

預告登記（土地法§79-1）制度目的為保全將來可能發生之請求權。借款擔保之正當工具為抵押權（民法§860以下）^[民法§860]。若以預告登記作為擔保替代工具，手段與制度目的間之對應關係，涉及民法第148條權利濫用禁止之判斷^[民法§148]。

**司法行政部（45）台公民字第523號函**（1956.01.31）確立重要原則：預告登記之效力不得對抗法院之查封行為。法院拍賣移轉產權與第三人時，預告登記即失其效力，無須撤銷。預告登記人如受有損害，可向不動產出賣人請求賠償。強調查封登記效力遠強於預告登記^[司法行政部45年函釋]。

### 二、當事人利益平衡（正當性）

若無實際借款關係，預告登記所保全之請求權是否存在及其範圍，為法律上之根本問題。同意書若係因被詐欺而簽署，依民法第92條得撤銷，須於發現詐欺後一年內為之（民法第93條除斥期間）^[民法§93]。

**最高法院92年度台上字第1309號民事判決**（2003.06.19）確立：預告登記同意書之簽署為法律行為，若所有權人係因被詐欺或脅迫而為同意，得依民法§92規定於發現詐欺後一年內（民法§93）撤銷其意思表示。撤銷後，該同意書自始無效，預告登記失所附麗，所有權人得請求塗銷。同時，請求權人如有詐欺行為，可能另涉刑法§339詐欺罪。實務上常見於民間二胎借款時，金主／代書利用借款人急迫輕率，使其以預告登記方式擔保非真實之債權^[最高法院92年度台上字第1309號判決]。

### 三、法條架構檢視（合規性）

土地法§79-1第1項以「有請求權存在」為前提。地政士法第27條第6款禁止明知不實而申辦登記^[地政士法§27-VI]。刑法第214條以「明知」為主觀要件，個案是否該當應由司法機關依具體事實判斷^[刑法§214]。

### 四、補充：借名登記與預告登記之糾葛

**最高法院108年度台上字第2162號民事判決**（2019.09.19）就借名登記關係中預告登記之效力有重要闡釋：借名登記關係中，出名人僅為名義登記人。若出名人未經借名人同意，擅自同意第三人辦理預告登記，而第三人明知或可得而知借名登記關係存在，該預告登記可能因違反誠信原則或構成權利濫用而被認定無效。借名人得主張終止借名登記關係後，請求返還土地，並同時請求塗銷預告登記^[最高法院108年度台上字第2162號判決]。

## 肆、上市櫃公司收受文件之實務框架

### 一、情境說明

實務上若上市櫃公司於業務過程中收受正本權狀、自然人憑證及印鑑證明，且當事人間無實際借款關係，此等文件組合之保管與使用，涉及多項法規之框架檢視。

### 二、各文件之法律地位

土地權狀為所有權證明文件。自然人憑證依內政部使用規範應由本人保管使用^[內政部憑證管理中心：moica.nat.gov.tw]。印鑑證明為辦理土地登記之重要文件。三文件若同時由他人持有，理論上具備辦理多種登記之文件條件，但實際登記仍須符合土地登記規則之各項要件^[全國法規資料庫]。

**內政部91年5月22日台內中地字第0910005955號函**補充：預告登記請求權人死亡時，其繼承人得繼承該預告登記所保全之請求權，並於原預告登記請求權人名義後加註繼承人姓名，無需重新辦理預告登記^[內政部91年函釋]。

### 三、相關法條框架

地政士法§27-6（明知不實申辦禁止）、§44（懲戒框架）^[地政士法§27-VI][地政士法§44]。刑法§214（使公務員登載不實）、§339（詐欺）、§358（入侵電腦），各以故意為前提^[刑法§214][刑法§339][刑法§358]。

證交法§20第1項禁止虛偽、詐欺等行為（含募集、發行、私募或買賣）；違反者依§171第1項第1款處3年以上10年以下有期徒刑^[證交法§20][證交法§171]。公司法§23規範負責人忠實義務與連帶賠償責任^[公司法§23]。

### 四、監理架構

上市櫃公司同時受金管會證期局（證券交易法）、地政主管機關（地政士法）、內政部（憑證管理）之規範^[證交法][地政士法][內政部]。投保中心團體訴訟機制在涉及多數受害人時可能啟動^[投資人保護法]。

## 伍、流程圖：借款→同意書→預告登記→塗銷→法院

```
【正常流程】
  借款人申請借款
       │
       ▼
  雙方協議擔保方式
       │
       ├── 抵押權設定（正常擔保方式，民法§860）
       │
       └── 預告登記（替代方式，土地法§79-1）
              │
              ▼
       所有權人簽署同意書
              │
              ▼
       向地政事務所申辦預告登記
              │
              ▼
       借款清償 → 權利人出具同意書 → 塗銷預告登記

【爭議流程 — 無實際借款】
  債務人遭勸誘簽署同意書
       │
       ▼
  代書辦理預告登記（可能涉及刑法§214）
       │
       ▼
  所有權人事後發現
       │
       ├── 民法§92撤銷同意（1年除斥期間）
       │      └── 請求塗銷預告登記
       │
       ├── 民法§767物上請求權
       │      └── 確認請求權不存在之訴
       │
       └── 法院查封拍賣（預告登記自動失效，45年函釋）
              └── 向出賣人請求損害賠償
```

**流程說明**：
1. **正常情形**：預告登記為借款擔保之替代工具，清償後塗銷
2. **爭議情形**：若無實際借款，同意書可能因詐欺而撤銷（最高法院92台上1309），或依民法§767請求塗銷（最高法院95台上2323）
3. **代書責任**：若代書明知不實仍辦理，可能構成刑法§214（臺灣高院110上易1167）
4. **查封優先**：預告登記效力不及於法院查封（司法行政部45年函釋）

## 陸、綜合整理與參考路徑

個案情形應依具體事實判斷，建議諮詢執業律師。

### 實務參考路徑

① 調閱土地登記謄本確認已登記事項 ^[全國地政電子謄本系統]
② 若涉及自然人憑證遭他人使用，可向內政部憑證管理中心洽詢 ^[內政部]
③ 若涉及地政士不當行為，可向地政士懲戒委員會檢舉 ^[地政士法§44~§48]
④ 涉及上市櫃公司事項，可向金管會證期局洽詢 ^[證交法§20]
⑤ 民事爭議得委任律師提出確認之訴或請求塗銷登記 ^[民訴§247][民法§767]
