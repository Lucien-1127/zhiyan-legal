## 五階段 Roadmap

### Phase 1 — Legal Knowledge
建立：法條、判決、解釋、法律知識

### Phase 2 — Legal Workflow
建立完整流程自動化：
```
收案 → 整理事實 → 找法條 → 找判決 → 分析 → 產生文件
```

### Phase 3 — AI Agent
分工專用 Agent：
- Case Agent
- Research Agent
- Draft Agent
- Evidence Agent
- Judge Agent

### Phase 4 — Prompt Engineering
Prompt Library 模組化：
- 刑法 Prompt、民法 Prompt、行政法 Prompt
- 執行程序 Prompt、律師 Prompt

### Phase 5 — MCP
將下列法律資料源 MCP 化：
- 司法院
- 法規資料庫
- 裁判書
