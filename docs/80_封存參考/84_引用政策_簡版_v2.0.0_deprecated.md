# 84_引用政策_簡版_v2.0.0_deprecated

## 核心規則

- 首次提及新來源用 …  
- 重複同來源不再加  
- 直接引文後加 [數字]  
- 目標：最小化 inline 標記，保持文章流暢

### 引用示例

| 版本 | 內容 |
|------|------|
| 舊版（密集干擾） | 水在常溫下沸點為 100℃（舊式S1），這是眾所皆知的事實（舊式S2）。 |
| 新版（清爽易讀） | 水在常溫下沸點為 100℃。水向下流動，形成瀑布。 |

---

## 引用時機

| 情況 | 做法 | 理由 |
|------|------|------|
| 明確數據／統計 | 加  | 必須溯源 |
| 事實陳述（一般知識） | 加 （第一次） | 追蹤來源出處 |
| 轉述對方主張 | 加 [數字] | 標記引述角色 |
| 自己推論 | 不加 | 邏輯推導，非引述 |
| 重複同一事實 | 不加 | 已在第一次標記 |

## 段落末尾引用表（C2.1）

```
【本段資料來源】
 標題 / 機構 — URL（或 Upload:檔名）
 標題 / 機構 — URL
```

### 完整段落示例

> 人工智能正在改變產業結構，89% 的企業已導入 AI 工具，未來五年將有 40% 的工作角色轉變。

```
【本段資料來源】
 2024 AI Adoption Report — https://example.com/ai-report-2024
 McKinsey Global AI Survey 2024 — https://example.com/mckinsey-ai-2024
 World Economic Forum Future of Jobs Report — https://example.com/wef-jobs-2024
```

## 全文末尾來源清單（C3）

### 使用時機
- 回答含 ≥3 段落，且每段不同來源

### 格式範本
```
【完整資料來源清單】

 標題 — 機構
    網址：URL
    更新時間：YYYY-MM-DD
    信度分級：[Official / Academic / Industry / Media]

 標題 — 機構
    網址：URL
    備註：上傳檔案可附檔名頁碼
```

### 信度分級
- **[Official]** 官方數據、政府文件、公司財報  
- **[Academic]** 學術期刊、研究機構  
- **[Industry]** 專業產業報告  
- **[Media]** 新聞媒體、評論文章  
- **[User-Provided]** 使用者上傳檔案

## 三種 MODE 適配規則（C4）

### MODE_QC（品質檢查）
```
1) 核心結論
2) 依據（分項）
3) 衝突檢查
4) 風險與邊界
5) 資料來源
```
- 引用於依據項後加   
- 末尾統一列「【資料來源】」

### MODE_RESEARCH（研究報告）
- 依據按官方→學術→媒體分層  
- 重點敘述加 [數字]  
- 末尾列「【完整資料來源清單】」含信度分級

### MODE_REPORT（正式報告）
- 依據段落落尾標註   
- 末尾列全清單  
- 超過 3000 字可分章節列表

## 特殊情況（C5）

### 上傳檔案引用
```
【本段資料來源】
 檔名 — Upload:檔案 UUID
    頁碼：第 X 頁
    引文：「...」
```

### Google Drive 檔案
```
【本段資料來源】
 檔名 — Space Files
    來源連結：Google Drive
```

### 多版本來源
```
 報告 v1.0 — URL
 報告 v2.0 — URL
```

## 衝突檢查標準（C6）

### 何為衝突
- 同一事實不同數據  
- 同一定義用詞差異  
- 不同時間導致結論相反

### 標註示例
```
【衝突檢查】
- 來源A：89% 企業採用 AI
- 來源B：65% 企業採用 AI
- 差異原因：定義不同
- 結論：採用來源B
```

## 品質檢查清單（C7）

| 項目 | 檢查點 |
|------|--------|
| inline 標記 | 首次引用才加 [數字] |
| 段落末尾 | 有引用必列「【本段資料來源】」 |
| 全文末尾 | 段落 ≥3 加「完整資料來源清單」 |
| 編號連貫 |  不跳號 |
| 信度標註 | REPORT 模式必須標註 |
| 衝突揭示 | 若有矛盾必說明 |
| 超連結可用 | URL 完整正確 |

---

## 常見問題（C8）

- **Q：能否省略來源？**  
  A：不能，即便是常識也需首次標註  

- **Q：多來源如何標註？**  
  A：依出現順序加數字，末尾列出完整清單  

- **Q：全文只有 2 個引用需完整清單嗎？**  
  A：不用，僅需段落末尾引用表
