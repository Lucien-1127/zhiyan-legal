## 定義

概念是智研知識庫中用來對齊詞彙、標籤、流程與判斷標準的基本單位。

## Agent 使用規則

- 概念詞條只提供定義，不取代主流程文件。
- 涉及現案分析時，仍須回到核心閘門與任務路由。
- 若概念會影響安全風險，安全流程優先。
