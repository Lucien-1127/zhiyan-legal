# 60_概念詞條_INDEX_v1.0.0

## 用途

本資料夾保存智研 AI Agent 的概念詞條。它用於補齊 Obsidian 內部概念連結，並讓 Agent 對法律、風險、事實分級、人格與流程詞彙有穩定定義。

## 使用規則

- 概念詞條只負責定義與對齊，不取代主流程文件。
- 現案分析仍須回到核心閘門與任務路由。
- 安全風險標籤一律優先轉安全流程。
- 新增 `[[概念]]` 時，應同步新增同名詞條。

## 詞條分類

| 類別 | 詞條 |
|---|---|
| 法域 | 刑法、民法、行政法、憲法、法律 |
| 事實控制 | 事實閘門、事實分級、事實摘要、局部不明警示、VERIFIED、CROSS_REFERENCED、USER_REPORTED、NEED_CHECK |
| 安全風險 | 安全、緊急、高風險求助、自傷、SELF_HARM、VIOLENCE、ABUSE、SCAM、PRIVACY、IMMINENT、INCIDENT |
| 任務流程 | 研究檢索、資料查核、報告產出、衝突檢查、審計層、四法要件表格評估 |
| 人格與部署 | 法律顧問人格、智研助教、Persona骨架、Space指令、Upload_檔名 |
| 工具治理 | 連結、SHA-256 Hash |

## 維護檢查

- 檔名型連結未解析數必須為 0。
- 概念型連結未解析數應盡量為 0。
- 不適合 Windows 檔名的概念，使用可用檔名加別名連結。
