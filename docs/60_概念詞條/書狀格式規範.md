# 書狀格式規範

## 適用範圍

民事訴訟書狀規則（pcode=B0010027，114年8月29日修正）、憲法法庭書狀規則、行政訴訟書狀規則。

## 核心條文

**民事訴訟書狀規則 §3**（全國法規資料庫聯網驗證）

### 紙張規格

| 項目 | 規範 |
|------|------|
| 尺寸 | A4（寬 21cm × 高 29.7cm） |
| 紙質 | 適於卷宗編訂及保存 |
| 列印 | 雙面列印（逾 30 頁須附目錄） |

### 版面配置

| 項目 | 規範 |
|------|------|
| 書寫方向 | 中文直式橫書，由左至右 |
| 邊界 | 上下左右各 **2.5cm** |
| 頁碼 | 頁面底端編列，置中 |

### 字型與行距

| 項目 | 規範 |
|------|------|
| 字體 | **標楷體**（TW-Kai，全字庫正楷體） |
| 字型大小 | **14pt ∼ 20pt**（實務以 14pt 為原則） |
| 行距 | 單行間距 或 固定行高 **25pt ∼ 30pt** |
| 行號 | 得於左側加行號（憲法法庭規則） |
| 框線/格線 | 電腦製作時不加（憲法法庭規則） |

### 附件

- 司法院 114/08/29 發布之參考格式附件一（民事書狀範本）
- 存放：`templates/民事書狀範本.docx`

## 系統實作

`src/zhiyan_legal/doc_generator.py` — `LegalDocument` 類別

### 使用方法

```python
from zhiyan_legal.doc_generator import LegalDocument

doc = LegalDocument()
doc.add_title('聲請延緩執行狀')
doc.add_reference('案號：XXX年度執字第XXX號')
doc.add_reference('受文者：○○地方法院檢察署')
doc.add_body('聲請事項：\n一、...\n二、...')
doc.add_section_title('事實及理由')
doc.add_body('...')
doc.save('/tmp/聲請狀.docx')
```
