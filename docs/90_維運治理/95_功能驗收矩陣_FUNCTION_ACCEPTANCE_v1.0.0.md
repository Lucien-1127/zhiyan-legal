# 98_功能驗收矩陣_FUNCTION_ACCEPTANCE_v1.0.0

## 驗收標準

本矩陣定義智研 AI Agent 上線前必須通過的功能。每個功能都要能被任務路由表識別，並能對應到一個主流程文件。

## 功能矩陣

| 功能 | 路由 | 主文件 | 必過條件 |
|---|---|---|---|
| 正式報告 | REPORT | `20_模式_REPORT_報告_v2.0.0` | 產出主管可讀結構，不混入教學語氣 |
| 研究整理 | RESEARCH | `21_模式_RESEARCH_研究_v2.0.0` | 區分來源層級與不確定性 |
| 品質查核 | QC | `22_模式_QC_查核_v2.0.1` | 列出問題、影響、修正建議 |
| 法律顧問 | CONSULTANT | `50_人格_顧問_v1.1.0` | 不保證勝敗，不補寫事實 |
| 申論批改 | TA_REVIEW | `51_人格_助教批改_v1.1.0` | 評結構、涵攝、攻防、法源 |
| 概念教學 | TUTOR | `52_人格_教學_v1.1.0` | 純教學，不提供現案建議 |
| 訴訟策略 | LITIGATION | `40_模組_訴訟策略_v2.2.0` | 先過核心閘門，再做攻防 |
| 安全風險 | SAFETY | `41_模組_安全風險對話處理_v1.0.0` | 安全步驟優先於法律分析 |
| 跨法域前置 | PRE_QC | `42_模組_Sentinel多法域前置檢測_v1.0.0` | 先判斷案型與法域，不急著輸出結論 |
| 引用政策 | CITATION | `30_引用政策_CITATION_POLICY_v2.0.0` | 單一引用格式，不載入簡版或舊版 |
| 概念詞條 | CONCEPT | `60_概念詞條_INDEX_v1.0.0` | 概念型連結可解析，不取代主流程 |

## 上線判定

| 等級 | 條件 |
|---|---|
| Ready | 結構測試通過，功能矩陣全數有主文件，路由案例全數可判斷，概念型連結可解析 |
| Limited Ready | 核心功能可用，但部分模組仍需人工確認 |
| Not Ready | 檔名型斷鏈、空檔、缺主文件、封存混入任一項存在 |

## 目前建議

智研可進入 `Ready` 上線測試。`42_模組_Sentinel多法域前置檢測_v1.0.0` 已改為 AI Agent 可直接使用的跨法域前置檢測規格，不再依賴程式碼草稿。
