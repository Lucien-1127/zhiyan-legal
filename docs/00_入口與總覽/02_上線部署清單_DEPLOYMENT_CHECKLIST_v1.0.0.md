- `80_封存參考` 全部排除主流程。

- `60_概念詞條` 可作輔助知識載入，但不取代主流程規則。

## 上線後監控

| 異常 | 處理 |
|---|---|
| Agent 回答太像聊天助理 | 檢查是否載入 `09_AGENT_SYSTEM_PROMPT` 與 MASTER |
| Agent 跳過查核直接結論 | 檢查 CORE_GATE 與 TASK_ROUTER |
| 引用格式混亂 | 只保留 `30_引用政策_CITATION_POLICY_v2.0.0` 作規則來源 |
| 舊版規則混入 | 確認沒有載入 `80_封存參考` |
| 高風險問題被一般回答 | 優先載入安全風險模組並回歸 T08 |
