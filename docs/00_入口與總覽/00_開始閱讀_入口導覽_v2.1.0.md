# 00_開始閱讀_入口導覽_v2.1.0

## 快速開始

1. 先讀本檔，確認智研資料夾結構。
2. 再讀 `01_README_總覽_v3.0.3` 與 `02_上線部署清單_DEPLOYMENT_CHECKLIST_v1.0.0`，理解哪些資料夾進主流程、哪些是封存、上線時該載入哪些檔案。
3. 啟動代理時，先讀 `09_AGENT_SYSTEM_PROMPT_v1.0.0`，再依序讀取核心控制層：`10_主人格_MASTER_v2.0.0`、`13_空間核心規格_PPL_SPACE_CORE_v3.0.0`、`11_啟動流程_BOOT_v2.40.0`、`12_核心閘門_CORE_GATE_v1.1.0`、`14_智研AI代理運行流程_RUNBOOK_v1.0.0`、`15_任務路由表_TASK_ROUTER_v1.0.0`。
4. 依任務選用模式、模組或人格。
5. 輸出前套用 `30_引用政策_CITATION_POLICY_v2.0.0` 與 `22_模式_QC_查核_v2.0.1`。
6. 上線前用 `95_冒煙測試_SMOKE_TESTS_v1.0.0` 與 `98_功能驗收矩陣_FUNCTION_ACCEPTANCE_v1.0.0` 驗收。

## 資料夾用途

| 資料夾 | 用途 | 載入建議 |
|---|---|---|
| `00_入口與總覽` | 使用入口、總覽說明、上線部署清單 | 先讀 |
| `10_核心控制層` | Agent System Prompt、主人格、啟動流程、核心閘門、空間核心、代理 Runbook、任務路由表 | 主流程必讀 |
| `20_模式與引用層` | REPORT / RESEARCH / QC 與引用政策 | 依任務載入 |
| `40_模組與人格層` | 訴訟策略、安全風險、Sentinel、顧問、助教、教學人格 | 依任務載入 |
| `60_概念詞條` | 法域、風險、事實分級、流程與部署概念 | 輔助載入 |
| `80_封存參考` | 舊版、簡版、參考檔 | 不進主流程 |
| `90_維運治理` | 清單、版本整理、淨化、路線圖、測試、匯出對照 | 維護時讀 |

| 品質查核 | `22_模式_QC_查核_v2.0.1` |

| 法律概念教學 | `52_人格_教學_v1.1.0` |
| 訴訟策略推演 | `40_模組_訴訟策略_v2.2.0` |
| 安全或高風險求助 | `41_模組_安全風險對話處理_v1.0.0` |

## 禁止事項

- 不從 `80_封存參考` 載入主規則。
- 不同時啟用互斥人格。
- 不在核心閘門前補寫事實。
- 不使用多份引用政策。
