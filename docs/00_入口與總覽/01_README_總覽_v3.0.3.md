# 01_README_總覽_v3.0.3

## 系統定位

智研是法律研究、品質查核與報告產出的 AI 代理知識庫。它的核心不是「一次讀全部」，而是依任務選擇必要層級，降低舊版規則、重複引用政策與人格漂移造成的干擾。

## 分層結構

| 層級 | 資料夾 | 內容 |
|---|---|---|
| L0 | `00_入口與總覽` | 使用入口與總覽 |
| L1 | `10_核心控制層` | 主人格、啟動流程、核心閘門、空間核心、代理 Runbook |
| L2 | `20_模式與引用層` | 三種模式與引用政策 |
| L3 | `40_模組與人格層` | 功能模組與人格模組 |
| Concept | `60_概念詞條` | 法域、風險、事實分級、流程與部署詞條 |
| Archive | `80_封存參考` | 舊版、簡版、參考檔 |
| Ops | `90_維運治理` | 文件治理、測試、匯出、路線圖 |

## 載入策略

- 一般任務：讀 `00_入口與總覽`、`10_核心控制層`，再按任務讀 `20` 或 `40`。
- 查核任務：額外讀 `22_模式_QC_查核_v2.0.1`。
- 高風險任務：優先讀 `41_模組_安全風險對話處理_v1.0.0`。
- 概念對齊：需要統一術語時讀 `60_概念詞條`。
- 維護任務：讀 `90_維運治理`，再視情況檢查 `80_封存參考`。

## 主流程排除

`80_封存參考` 只保留歷史與對照，不作為主規則來源。若內容需要回流主流程，必須先合併到 `10`、`20` 或 `40` 的主檔。

## 維護檢查

- 改名或移動後，檢查檔名型 `[[連結]]`。
- 新增主檔後，更新 `96_匯出對照_EXPORT_MAP_v1.0.0`。
- 改變載入順序後，更新 `14_智研AI代理運行流程_RUNBOOK_v1.0.0`。
- 每輪整理後，跑 `95_冒煙測試_SMOKE_TESTS_v1.0.0`。
