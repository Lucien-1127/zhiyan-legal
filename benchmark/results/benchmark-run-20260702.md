### 申論題一：違法搜索與毒樹果實（★★★★★）

**事實：** 警方無搜索票進入甲住宅搜索，查獲毒品及手機，再依手機內容查獲共犯乙。

**答案摘要：**
| 子題 | 結論 | 關鍵引用 |
|:-----|:-----|:---------|
| Q1 無令狀搜索是否合法？ | 不合法 — 違反§128令狀原則，無§131急迫要件 | §128, §131 |
| Q2 毒品證據能力？ | 應依§158-4權衡 — 違法情節重大，傾向排除 | §158-4, 93台上664號[A] |
| Q3 手機資料證據能力？ | 同毒品，與違法搜索具直接因果關係 | §158-4, 102台上3254號[B] |
| Q4 共犯乙是否屬毒樹果實？ | 是 — 「依手機內容查獲」非獨立合法程序 | 105台上3011號[C] |
| Q5 法院應如何判決？ | 排除後若無其他證據→§301無罪判決 | §301 |

**推理鏈：** 違反告知義務 → 程序違法（無令狀）→ 不正方法 → §158-4權衡 → 毒樹果實一體排除 → 證據不足無罪

**評分：** 🅐（precedent_grade: A，引用分級明確）

### 申論題二：警詢自白（★★★★★）

**事實：** 警方說「你現在不承認，以後法官一定判更重」，甲因此自白，審判中翻供。

**答案摘要：**
| 子題 | 結論 |
|:-----|:-----|
| Q1 是否違反緘默權？ | **足以認定具有不正訊問之重大疑義**（非「一定違反」） |
| Q2 是否違反任意性？ | 是 — 以未來量刑施壓，可能構成不正方法 §98 |
| Q3 自白證據能力？ | 原則排除 — 優先§158-2（違反告知義務），次之§156（不正方法） |
| Q4 可否作為補強證據？ | 若自白已排除，不得作為補強證據 |

**關鍵引用：** §95, §98, §156, §158-2, 釋字第582號

### 申論題三：正當防衛（★★★★☆）

**事實：** 甲見乙持刀追趕丙，持木棍攻擊乙致死亡，事後發現乙為便衣警察。

**答案摘要：**
| 子題 | 結論 |
|:-----|:-----|
| Q1 正當防衛？§23 | 不成立 — 乙為依法執行職務（§21），非不法侵害 |
| Q2 誤想防衛？ | 成立 — 甲主觀誤信不法侵害，依嚴格罪責理論阻卻故意 |
| Q3 過失？§14, §276 | 視情境 — 若已盡注意義務則不成立；反之成立過失致死 |
| Q4 不知乙身分之影響？ | 誤想防衛核心要素，阻卻故意，僅論過失 |

**關鍵引用：** §21, §23, §14, §276, 最高法院102台上3895號

### 申論題四：共同正犯（★★★★★）

**事實：** 甲說「去教訓他一下」，乙說「好」，乙持刀殺害丙。

**答案摘要：**
| 子題 | 結論 |
|:-----|:-----|
| Q1 共同正犯？§28 | 不成立殺人共同正犯 — 乙持刀殺人屬「實行過限」 |
| Q2 教唆犯？§29 | 成立傷害教唆犯 — 對殺人結果不負責 |
| Q3 幫助犯？§30 | 不成立 — 甲屬「造意」非「輔助」，教唆吸收幫助 |
| Q4 犯意聯絡判斷？ | 明示/默示/間接皆可；責任範圍以犯意聯絡為界 |

### 申論題六：跨程序綜合題（★★★★★★）

**事實：** 拘提→請求律師被拒→保持沉默→警方說「不講話就是沒悔意」→自白→搜索手機→查獲銀行帳戶→找到共犯→法院再次自白

**答案摘要：**
| 子題 | 結論 |
|:-----|:-----|
| Q1 辯護權侵害？ | 是 — 警方拒絕等待律師，違反§27, §31⑤ |
| Q2 緘默權侵害？ | **足以認定具有不正訊問之重大疑義** |
| Q3 自白證據能力？ | 原則無 — §158-2（違反告知）+ §156（不正方法） |
| Q4 手機資料證據能力？ | 應依§158-4權衡 |
| Q5 銀行帳戶證據能力？ | 毒樹果實衍生證據，§158-4權衡 |
| Q6 共犯供述受影響？ | 若源自違法手機內容，同受影響 |
| Q7 法院自白補正？ | 需審查四因素：時間間隔、律師協助、權利告知、自由意志 |
| Q8 排除後仍可能有罪？ | 是 — 若有其他合法證據（非衍生），仍可定罪 |

**推理鏈：** 拘提→違反告知義務→§158-2先適用→不正方法§156→排除自白→毒樹果實→手機/銀行/共犯依§158-4個案權衡→法院自白需嚴格審查稀釋效應→有其他證據≠無罪
